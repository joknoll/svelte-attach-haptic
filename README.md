# svelte-attach-haptic

Add haptics to svelte components.

**Example**

```svelte
<script lang="ts">
</script>

<button {@attach haptic()}>
</button>
```

For a more advanced example look at the [playground](/playground/src/routes/+page.svelte).

[Demo](https://knolljo.github.io/svelte-attach-haptic/)
